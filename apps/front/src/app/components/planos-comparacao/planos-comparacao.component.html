Como entender os limites?

📁 Formulários = Sites diferentes

Cada site que você automatiza conta como 1 formulário. Por exemplo:

  • • Formulário de vaga (20+ campos) = 1 formulário
  • • Cadastro de site (15+ campos) = 1 formulário
  • • Login de plataforma (10+ campos) = 1 formulário
  • • Formulário de contato (8+ campos) = 1 formulário

📄 Campos = Informações que você preenche

Cada campo de formulário conta para o limite total:

  • • Nome = 1 campo
  • • Email = 1 campo
  • • Telefone = 1 campo
  • • Endereço = 1 campo
  • • Experiência = 1 campo
  • • Formação = 1 campo
  • • Habilidades = 1 campo
